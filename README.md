# Projeto LP2 2024/2025 - The Night of the Living Deisi

🎮 Jogo desenvolvido em Java no âmbito da unidade curricular de Linguagens de Programação II (LP2).

O projeto tem como principal objetivo aplicar, de forma prática, os conceitos fundamentais de Programação Orientada a Objetos, incluindo encapsulamento, herança, polimorfismo e organização modular do código.

A lógica do jogo é estruturada através de um modelo orientado a objetos, promovendo separação de responsabilidades, reutilização de código e extensibilidade da aplicação.

A interface gráfica foi fornecida pelos docentes, sendo o foco principal do trabalho a modelação do domínio e a implementação da lógica do jogo.
